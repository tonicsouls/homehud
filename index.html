Each module is a self-contained directory that can be independently deployed to your SenseCraft HMI display. Each module contains:

  • index.html - Documentation and setup guide
  • data/ - Source data files (words, articles, etc)
  • templates/ - SenseCraft JSON templates for the display

To use any module in SenseCraft, simply copy the template URL (hosted on GitHub Pages) into the Web Content feature.
